FAQ

Does Liveboot modify my disks?

No. It runs entirely in RAM by default. Tools that write to disks ask for confirmation and are opt‑in.

What hardware is supported?

Anything a standard Linux kernel supports. We target modern x86‑64 and common Wi‑Fi/Ethernet chipsets. ARM builds may follow.

How big is the image and how fast does it boot?

We aim for a small footprint with fast, network‑enabled boots. Exact sizes depend on enabled toolsets per release.

Can I customize the toolset?

Yes. The NixOS base makes it straightforward to build custom variants or enable extra modules.

Is Secure Boot supported?

Planned. We’re working on signed artifacts and reproducible builds; roadmap will note progress.
